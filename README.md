# QIANKUN 微应用解决方案 Demo

1. 克隆项目到本地

```base
$ git clone https://github.com/humandetail/qiankun-demo.git
```

2. 进入主应用，安装依赖，启动应用

```base
$ cd main-app
$ npm i
$ npm run serve
```

3. 进入次应用，安装依赖，启动应用

```base
$ cd micro-app-1
$ npm i
$ npm run serve

$ cd micro-app-2
$ npm i
$ npm run serve


$ cd micro-app-1-1
$ npm i
$ npm run serve
```

4. 访问 [http://localhost:8080](http://localhost:8080) 来查看效果
